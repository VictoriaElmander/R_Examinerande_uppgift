# Analys av försäkringskostnader

Detta projekt analyserar vilka faktorer som är relaterade till 
försäkringskostnader samt att utvärdera om en regressionsmodell kan användas 
för att förklara och förutsäga kostnader för framtida kunder.

## Filer
`scripts/` innehåller alla skript som används i analysen
`report/` innehåller rapporten (report.qmd)
`data/` innehåller dataset (insurance_costs.csv)
`run_analysis.R` kör hela analysen (exkluderar filer i work/)


## Så kör du projektet
Öppna projektet i RStudio och kör `run_analysis.R`

## Paket
`tidyverse`
`lmtest`
`patchwork`
`here`

## Output
Färdig rapport genereras från `report/`
